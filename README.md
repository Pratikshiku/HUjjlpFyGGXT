### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 校园课表
5. 教室信息
6. 课程信息
7. 校园资讯
8. 后台管理
9. 个人中心
10. 我的收藏


**教师功能：**
1. 登录
2. 注册
3. 首页
4. 校园课表
5. 教室信息
6. 课程信息
7. 校园资讯
8. 后台管理
9. 个人中心
10. 我的收藏



**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 学生管理
4. 教师管理
5. 校园课表管理
6. 教室信息管理
7. 课程信息管理
8. 系统管理（轮播图管理、校园资讯）
